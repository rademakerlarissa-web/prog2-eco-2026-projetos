## Sobre este repositório

Este repositório organiza todas as entregas da disciplina de Programação II ao longo do semestre.
Aqui você encontra:

- **`/projetos`** – projeto semestral desenvolvido em duplas, evoluído incrementalmente.
- **`/exercicios`** – atividades e listas de exercícios de cada aula.
- **`/templates`** – modelos de README e estrutura de projeto para as duplas usarem.

---

## Estrutura do repositório

```
prog2-eco-2026-projetos/
│
├── projetos/
│   ├── dupla-01-nome1-nome2/
│   │   ├── README.md        ← descrição, integrantes e link do projeto
│   │   └── src/             ← código-fonte
│   ├── dupla-02-nome1-nome2/
│   └── ...
│
├── exercicios/
│   ├── 9-revisao-oo/
│   │   ├── enunciado.md     ← disponibilizado pela professor
│   │   └── entregas/
│   │       ├── nome-Estudante1/
│   │       └── nome-Estudante2/
│   ├── aula-02-ide-git/
│   └── ...
│
└── templates/
    ├── README-dupla.md      ← modelo de README para o projeto da dupla
    └── README-exercicio.md  ← modelo de README para exercícios
```
